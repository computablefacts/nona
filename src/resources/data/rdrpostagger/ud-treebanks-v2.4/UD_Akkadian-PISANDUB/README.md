# Summary

A small set of sentences from Babylonian royal inscriptions.


# Introduction

‘PISANDUB’ means ‘basket of tablets’ (Sumerian logogram for Akkadian pisanduppu
/ pisan ṭuppātim), which reflects the scale of this project.
